## LinkedIn

AI coding agents write great application code. They're terrible at infrastructure.

Today we're releasing **Tigris Agent Plugins** — a plugin that teaches your AI coding agent how to use Tigris. It ships:
- **5 CLI skills** covering auth, buckets, objects, access keys, and IAM — each loads automatically when the conversation needs it
- **A storage subagent** for multi-step workflows like project setup, S3 migration, dev sandboxes, and security audits
- **2 security rules** that enforce private buckets, scoped keys, snapshot-before-delete, and credential safety by default

Works with Claude Code, Cursor, OpenCode, OpenAI Codex, and Pi. Install the plugin, say "set up a bucket for my new project," and the agent knows exactly what to do — no docs, no guessing at flags.

Pairs with the Tigris MCP server for full tool access. Open source under MIT.
