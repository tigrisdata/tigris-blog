Happy to announce that [Next.js][nextjs-url] now has an officially supported example for bootstrapping
your projects using [Tigris][tigris-url]. Creating a project with Next.js and Tigris is as simple as:

```shell
$ npx create-next-app --example with-tigris
```

<!--truncate-->

## Getting started

Next.js allows you to bootstrap Next.js apps using [create-next-app](https://nextjs.org/docs/api-reference/create-next-app)
utility.

### Install Tigris

- For **macOS**: `brew install tigrisdata/tigris/tigris-cli`
- Other operating systems: [See installation instructions here](https://docs.tigrisdata.com/cli/installation)

### Startup Tigris dev environment

```shell
$ tigris dev start
```

### Initialize your Next.js app

Use the official [create-next-app](https://nextjs.org/docs/api-reference/create-next-app) utility

```shell
$ npx create-next-app --example with-tigris
```

Run the app and see it will automatically create your databases and collections.

```shell
$ cd my-tigris-app
$ npm run dev
```
